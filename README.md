# Cavo Zoe — монитор освободившихся номеров

Опрашивает скрытый JSON-эндпоинт бронирования Cavo Zoe и присылает в Telegram
всем указанным пользователям уведомление, как только на **22–25 июля 2026**
(заезд 22-е, 3 ночи, 2 взрослых, 1 номер) появляется свободный номер.

## Как это работает
- `POST https://cavozoe.reserve-online.net/avl` возвращает JSON с таблицей номеров.
- Каждая ставка помечена `data-status`: `AVL` = свободно, `SS` = занято.
- Если появляется хоть одна `AVL` ставка (или `price_check.best > 0`) → шлём алерт.
- Состояние хранится в `state.json`, поэтому спама нет: сообщение уходит
  при появлении номера и повторяется не чаще, чем раз в 30 мин, пока он свободен.

## Настройка (один раз)

### 1. Создать Telegram-бота
1. В Telegram открой **@BotFather** → команда `/newbot`.
2. Задай имя и username → BotFather выдаст **токен** вида `123456789:AAE...`.
3. Вставь токен в `config.json` → поле `telegram_token`.

### 2. Узнать chat_id обоих получателей
1. Оба пользователя открывают своего бота и жмут **Start** (или пишут любое сообщение).
2. Запусти:
   ```bash
   /Users/maksymkogua/miniconda3/bin/python3 get_chat_ids.py
   ```
3. Скопируй показанные id в `config.json` → массив `chat_ids`, например:
   ```json
   "chat_ids": [111111111, 222222222]
   ```

### 3. Разовая проверка
```bash
/Users/maksymkogua/miniconda3/bin/python3 check.py
```
Сейчас на 22–25 июля всё занято → в логе будет `Sold out`. Чтобы убедиться,
что Telegram работает, можно временно поставить `"checkin": "2026-09-15"`
(там есть места) — придёт тестовое сообщение. Не забудь вернуть `2026-07-22`.

### 4. Запустить автопроверку каждые 3 минуты
```bash
bash install.sh
```
Остановить:
```bash
bash uninstall.sh
```

## Чтобы Mac не засыпал (ночью проверки идут только если Mac бодрствует)
Запусти в отдельном окне терминала и оставь:
```bash
caffeinate -s
```
или системно (до перезагрузки):
```bash
sudo pmset -a sleep 0
```

## Файлы
- `check.py` — сам монитор (только стандартная библиотека Python).
- `config.json` — даты, токен, chat_ids.
- `get_chat_ids.py` — помощник для получения chat_id.
- `com.cavozoe.monitor.plist` — задание launchd (каждые 180 с).
- `install.sh` / `uninstall.sh` — установка/удаление задания.
- `logs/monitor.log` — журнал проверок.
